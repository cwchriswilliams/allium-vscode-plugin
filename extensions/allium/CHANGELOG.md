# Changelog

All notable changes to the Allium VS Code extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-18

### Added
- Initial pre-release of Allium Tools for VS Code.
- Language Server Protocol integration for robust analysis.
- Syntax highlighting for all Allium language constructs.
- Core editor features: Go to Definition, Find References, Document Symbols.
- Diagnostics for logic errors, missing clauses, and undefined references.
- Safe Fixes for common specification issues.
- Refactorings: Rename (workspace-wide), Extract Literal, Extract Enum.
- Advanced Tools: Diagram Preview (D2/Mermaid), Rule Simulation, and Test Scaffolding.
- Document formatting support.
- Snippets for all top-level declarations.
